# Java并发编程面试题

## 一、多线程基础

### 1. 进程和线程的区别？

​		进程就是程序的一次执行过程，线程是进程中的⼀个执行单元，负责当前进程中程序的执行，⼀个进程中至少有⼀个线程。**进程是操作系统`资源分配`的基本单位，而线程是`处理器（CPU）任务调度和执行`的基本单位**。

​		由于同一进程中的的线程共享本进程的地址空间和资源，而进程之间的地址空间和资源是相互独立的。所以一个进程崩溃后，并不会对其他进程产生影响，但是一个线程崩溃会导致整个进程都死掉。所以多进程要比多线程健壮。
